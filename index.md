# Експлуатаційна Документація ППУ-М

Ласкаво просимо до технічної документації **пересувного пункту управління (ППУ-М)**.

Цей посібник містить важливі інструкції, обмеження та умови роботи, необхідні для безпечної та безперебійної експлуатації комплексу.

Використовуйте бічну панель навігації для переходу між розділами.

**⚠️ УВАГА:** Перед початком роботи обов'язково ознайомтеся з усіма наведеними інструкціями.
